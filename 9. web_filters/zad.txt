Задачи:
1. Инсталирайте node.js
2. Напишете в CMD командите: node/ nodejs/ npm
3. Инсталирайте npm (Node package manager)
4. Създайте папката web_filters
5. В нея инсталирайте с npm пакета express




Команди в cmd:
cd 	печата текущата работна директория
cd .. 		връщаме се в родителксата директория
cd papka	влизаме в папката с име papka
dir		печата съдържанието на текущата работна директория
npm 		node package manager. програма, която менижира/инсталира/трие/създава пакети на node.js
npm init	създава нов node.js проект в текущата работна директория
npm install pesho		опитва се да инсталира пакета pesho

CTRL + C 	спира програма, която работи  в терминала

node server.js  пускаме сървъра, 
написан в файла server.js

Стъпки за setup:
1. Направих папката web_filters
2. Влязох в тази папка през CMD (cd)
3. Създадох празен проект (npm init)
4. Инсталирах express (npm install express)
5. В тази папка направете файла server.js


Задачи:
1. Подкарайте простия web server
2. Направете така, че когато заредим
localhost:3000/ да се зарежда файла
nachalna_stranica.html
* напишете сами файла nachalna_stranica.html
* заредете картинка в html файла
3. Направете папката public и сложете няколко
картинки в нея
4. Намерете начин (в нета) да пращаме
автоматично всеки файл, който бъде поискан
от папката public
* гугълнете express static file serving

Домашна работа:
1. Направете начална страница, която се зарежда 
при посещаване на :
localhost:3000/

Началната страница напишете в файла 
index.html

Нека в началната страница се зарежда картинка

2. Проучете как автоматично да се 
зареждат всички файлове, сложени в папката public
(автоматично означава да нямаме app.get за всеки
файл, който искаме да може да се зарежда)

* проверете в интърнет как се прави
express static file server
